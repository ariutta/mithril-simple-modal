# Testing

### Compile the code:

Create dir `./test/lib/mithril-simple-modal/dev`

```bash
npm compile
```

### Run Unit Tests

```bash
npm tests
```

### Run End-to-End Tests

Currently, they are manual only. TODO: set up selenium.

```bash
npm start # start test server
```

Visit [test selection page](http://localhost:8080/test/).
